Mobile First

  • Iš mažesnio ekrono dydžio yraa lengvesnis perėjimas į didesnį.(lengviau pridėti naujo turinio nei atimti seną.
  • Kocentracija į turinį(dėl mažo vietos kieko)
  • Didesnis kiekis vartotojų naudoja moviliasias aplikacijas.(Orientojamies į didesnį vartotojų kieki)
  • Google paieškos alogritmas geriau įvertina Mobile First kurtas svetaines.

Naudojant Mobile First dizaino principą, pradedame nuo mažiausio ekrano ir judame link didesnių Tan naudojame @media (min-width __px)
