Tėvinis elementas, gali turėti daugelį vaikinių elementų. Vaikinis elementas, gali turėti tik vieną tėvinį elementą.

Paveldėjimas

Paveldėti stilius, gali tik vaikinis elementas iš tėvinio elemento.
Paveldimi stiliai yra tie kurie keičia elemento išvaizdą, ne būseną. PVZ : color, font-size, font-family...
Nepaveldimi stiliai yra tie, kurie keičia elemnto būseną. PVZ: width, height, display, border, pasition...
Net paveldėti stiliai, gali nebūti atvaizduojami dėl stilių prioritetų.

Numatytieji stiliai

Numatietiji stiliai yra automatiškai naršyklės suteikti stiliai. Tarp skirtingų naršyklių numatytieji stiliai gali minimalistiškai keistis ir šitaip sugadinti sveteinės vaizdą skirtingų naršyklių vartotojams.
Tam, kad nebūtų problemų kuriant dizianą skirtingoms naršyklėms, yra naudojimi Reset arba Nomalize CSSai.
Normalize - suvienodina visus stilius tarp naršyklių ir panaikina dalį kenkėjiškų stilių.

CSS Prioritetai

  1. Inline stilius - 1000
  2. ID- 100
  3. Class - 10
  4. Element - 1
  5. CSS poins
  6. CSS Order(in CSS code)
  7. Inheritence
    8. Google

Esant dviem slektoriam, kurie nusako to paties elemnto voenodus stilius, yra atsižvelgiama į tų selektorių prioritetus. Aukščiau esantys prioritetai, turi aukštesnį prioritetą.
Esant dviem vienodo priereitetų balo selektoriams, kiečiant tą patį elemento stilių, yra atsižvelgiama į CSS eliškumą. Kuris CSS stilius aprašytas vėliau, tas ir bus naudojamas.

Jeigu įmanomą - nenaudoti !!!!important prierašas ignoruoja visus prioritetus ir suteikia tą stilių, apart kito !important prierašo, kuris turi aukštesnį prioritetą.

Icons

Ikona yra mažas paveiksliukas.

Iconos stilizuojamas kaip tekstas. Nepriskiariame width/height, priskiariame font-size. Nepriskiriame backgraund-color, prisikiriame color...
