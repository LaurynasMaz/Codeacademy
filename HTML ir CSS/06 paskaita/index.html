Psiaudo Selektoriai

  • *
  • element
  • class
  • #id
  • selector1, selector2
  • selector1 selector2
  • selector1 > selector2
  • element.class#id- elementas turinitis nurodytą klasę/es ir arba nurodyta id
  • selector1 + selector2 - iš karto selector1 einantis selector2
  • selector1 ~ selector2- už selector1 visi einatys selector2
  • [attribute] - atribudas
  • [attribute=value]- atribudas turintis nurodytą reikšmę
  • [attribute~=value]- atribudas kurio reikšmėje yra nurodytas žodis
  • [attribut*=value]- atribudas kurio reikšmėje yra nurodyto žodžio dalis
  • [attribute|=value]- atribudas kurio reikšmė yra value arba value kažkas.
  • [attribute^=value]- atribudas kurio reikšmė prasisdeda value ...
  • [attribute$=value]- atribudas kurio reikšmė pasibaigia value
  • :link- paprasta/nelankyta nuoroda
  • :visited- lankyta nuoroda
  • :hover- užėjės kursorius
  • :active- aktyvi (paspaudimo momentu) nuoroda
  • :element:empty- tusčias elementas (be content)
  • ::marker - sąrašo elemetų žymės
  • ::not(p) - neparenka nurodyto elemento
  • :first-child - parenka pirmajį vaikinį elementą
  • :last-child - parenka paskutinijį vaikinį elementą
  • :only-child - parenka vienitelį vaikinį elementą(jeigu jis yra vienas)
  • :first-of-type - parenka pirmajį nurodyto tipo elementą
  • :last-of-type - parenka paskutinijį nurodyto tipo elementą
  • :only-of-type - parenka vienitelį nurodyto tipo elementą
  • :nth-child(n) - parenka nurodytus vaikinį/us elementą/us
  • :nth-last-child(n)parenka nuo galo nurodytus vaikinį/us elementą/us
  • :nth-of-type(n)- parenka nurodytą/us elementą/us pagal tipą
  • :nth-last-of-type(n)- parenka nuo galo nurodytą/us elementą/us pagal tipą
  • marker - sąrašo elementų žymės
  • ::before - sukuria turinį prieš elementą
  • ::after - sukuria turinį už elementą
  • input:checked - pažymetus/ parininktus radio/checkbox/select pasirinikimus
  • input:default - iš anksto parininktas pasirininkimas.
  • input:disabled - inputas, kuris turi disabled atributą
  • input:enabled - inputas, kuris neturi disabled atributo
  • input:focus - inputas, ant kurio yra paspausta/fokusuota.
  • input:in-range - inputas, kurio vertė yra nurodytose ribose.
  • input:out-of-range - inputas, kurio vertė yra už norodytų ribų.
  • input:inderterminate - inputas, kurio reikšmė neapibrėšta.
  • input:invalid - inputas, kurio vertė netinkama
  • input:valid - inputas, kurio vertė tinkama
  • input:required - inputas, su required atributu.
  • input:optional - inputas, ne required atributo.
  • input:read-only - inputas, su read-only atributu.
  • input:read-write - inputas, ne read-only atributo.
  • input::placeholder - Įvesties laukelio placeholder atributo reikšmės parinkimas stilizavimui.
  • element::first-letter - parenka elemento pirmają raidę
  • element::first-line - parenka elemento pirmają eilutę
  • :selection - parenka vartotojo pažymetą infomaciją.
  • is()
  • has()
  • where()
