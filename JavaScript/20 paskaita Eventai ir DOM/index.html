Events

Įvykiai - puslapyje vykstanties kažkokie veiksmai/įvykiai (pelės paspaudimas, klaveturos pasapudimas, informacijos pažymėjimas, scrolinimas, forms pateikimas, failo įkėlis/atsisiuntimas...)
Norėdami klausyti ar nutinka kažkoks įvykis, turime pražioje nurodyti(selektinti) kurioje DOM dalyje tas įvykis gali nutikti.
Nusirodžius(selektinus), kurioje dalyje klausomės įvykio - reikia aprašyti įvykio klausymąsi. Tai padarome rašydami "addEventListener" metodą, kuris piriima 2 parametrus: koksĮvykis, kąDarytiNutikusĮvykiui.

Document Object model

Dokumentas- puslapis kuriame yra visa vartotojui atvazduojama informacija (tekstas, paveiksliukas, fomra lentelė...)
Obejktas - kažoks puslapio elementas(nuo html tago, iki ikonos ar span tago)
Modelis - objektų stuktūra kazkuriame elemente(tekstas, paveiksliukas, fomra lentelė...)

Norėdami per JS selektinti kažkurią DOM daį turime rašyti tokį kodą: "document.querySelector" arba "document.querySelectorAll". "querySelector" priima 1 parametrą: - selektorių (tokį kokį rašėme CSS)
